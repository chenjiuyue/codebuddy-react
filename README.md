# CodeBuddy React App

一个简单而现代的 React 应用，使用 Vite 构建。

## 特性

- ⚡ 使用 Vite 构建，享受极速的开发体验
- 🎨 现代化的 UI 设计
- 🔧 简洁的项目结构，易于扩展

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:3000` 打开。

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 项目结构

```
codebuddy-react/
├── src/
│   ├── App.jsx          # 主应用组件
│   ├── App.css          # 应用样式
│   ├── main.jsx         # 应用入口
│   └── index.css        # 全局样式
├── index.html           # HTML 模板
├── package.json         # 项目配置
└── vite.config.js       # Vite 配置
```

## 技术栈

- React 18
- Vite 5
- CSS3
